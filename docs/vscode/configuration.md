# Configuración

## Plugins Útiles

- Python
- Vetur
- Docker
- YAML
- markdownlint
- Djaneiro - Django Snippets

## Configuración del Editor

### Configuración workspace

```json
"python.linting.flake8Enabled": true,
"[python]": {
    "editor.rulers": [79, 120]
}
```

### Configuración Usuario

Cambiar el valor *Render Whitespace* a *all*
